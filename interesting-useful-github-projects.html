Interesting & Useful Github Projects

Aug 23, 2020  |  Ashish Doneriya  |   Lists   Software Engineering  

Tools

  • Optimus - Image conversion and optimization desktop app.

  • vidmerger - A wrapper around ffmpeg which simplifies merging multiple videos

  • eleventy - A simpler static site generator. An alternative to Jekyll. Transforms a directory of templates (of varying types) into HTML.

  • Ballista: Distributed Compute Platform - Distributed compute platform implemented in Rust, using Apache Arrow memory model.

  • Supabase - The open source Firebase alternative. Supabase adds realtime and restful APIs to Postgres without a single line of code.

  • BTFS - A bittorrent filesystem based on FUSE.

  • FXFaceDetection - Real-Time Face Detection App using Computer Vision & JavaFX
